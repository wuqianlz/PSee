# PSD / PSB 文件查看器开发设计文档

## 1. 项目概述

本项目旨在实现一个类似 Photoshop 的轻量级 PSD / PSB 文件查看工具，提供基础解析与可视化能力，适用于设计预览、图层分析等场景。

### 1.1 核心功能

- 支持本地选择 `.psd` / `.psb` 文件
- 解析文件结构（图层、蒙版、文本、样式等）
- 渲染最终合成图像
- 提供图层面板（类似 Photoshop）
- 支持图层显示/隐藏、透明度展示、层级结构查看

### 1.2 优化空间

- 后续支持图像编辑能力（避免复杂度过高）
- 后续支持 PSD 文件写回（扩展）

---

## 2. 技术选型

### 2.1 方案

| 模块 | 技术选型 |
|------|----------|
| 前端框架 | Vue3 + TypeScript |
| 构建工具 | Vite |
| PSD解析 | ag-psd |
| 渲染引擎 | Canvas |
| 状态管理 | Pinia |

---

### 2.2 方案

| 方案 | 优点 | 缺点 |
|------|------|------|
| Vue + TypeScript解析 | 架构简单、无需后端、支持本地文件、交互流畅 | 大文件（PSB）性能有限 |

---

## 3. 系统架构设计

### 3.1 整体架构

```text
